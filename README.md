# Nautitech Immersive Landing Page

Vite + React immersive one-page experience for Nautitech.

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Edit content

Most editable content lives in:

- `src/data/siteContent.js`
- `src/data/events.js`
- `src/data/dimensions.js`
- `src/data/specs.js`

## Scroll behavior

Desktop snap scroll is controlled in `src/data/siteContent.js`:

```js
snapScroll: true
```

Mobile keeps natural scroll.

## Assets

Main assets live in `public/assets/`.
